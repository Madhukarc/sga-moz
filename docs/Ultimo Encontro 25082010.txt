2- Logs
Criação de LOG's para registo de todas as operações de mudança de registo de notas
 
3- Formulários 
Manter o mesmo formulário para criação/alteração e consulta fazendo apenas o bloqueio dos campos no modo consulta, desta forma o impacto para o utilizador é menor.
 
4- Menus
Realçar a opção de menu seleccionada.
 
5- Utilizadores
Mostrar qual o utilizador logado e colocar uma opção de logoff no cabeçalho do ecran (por cima dos tab's dos menus).
No ecran de login colocar uma opção de alterar password.
 
6- Listas
Quando se selecciona um tipo de informação deverá ser mostrada a lista, na lista deve aparecer um botão imprimir e ao clicar deve surgir a opção <tudo> ou pedir os limites inferior e superior (critério de seleccção).
 
7 - Alertas
Possibilidade de definir alertas automáticos.
P.Ex : 
- Informar aos docentes a data limite de lançamento de notas (se não for introduzido até à data é inactivada a opção para o docente e activada para a secretaria).
- Informar os estudantes das datas de exame/entrega de trabalhos.
 
8- Codificação de Alunos/Turmas
Ver se há alguma regra definida para se poder criar uma opção de configuração, a sugestão é <ANO>+<num.sequencial>
Pinki ficou de validar se há ou não regras já definidas na instituição.
